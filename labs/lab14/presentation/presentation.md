# Цель работы

## Основная цель

Получить навыки создания разделов и файловых систем, а также монтирования в ОС Linux.

# Ход выполнения работы

## Создание разделов MBR с помощью fdisk

- Добавлены два диска: **disk1** и **disk2**
- Создан основной раздел **/dev/sdb1**

![Создание MBR-раздела](Screenshot_1.png){ #fig:001 width=70% }

## Создание расширенного и логического разделов

- Добавлен расширенный раздел **/dev/sdb2**
- Создан логический раздел **/dev/sdb5** размером 300 МиБ

![Создание логического раздела](Screenshot_5.png){ #fig:002 width=70% }

## Создание раздела подкачки (swap)

- Создан раздел **/dev/sdb6**  
- Изменён тип раздела на **82 (Linux swap)**  

![Создание раздела swap](Screenshot_7.png){ #fig:003 width=70% }

## Создание разделов GPT с помощью gdisk

- Для диска **/dev/sdc** создана таблица **GPT**
- Добавлен раздел **/dev/sdc1** размером 300 МиБ, тип **8300 (Linux filesystem)**

![Создание раздела GPT](Screenshot_10.png){ #fig:004 width=70% }

## Форматирование файловых систем

- **/dev/sdb1** отформатирован в **XFS**  
- **/dev/sdb5** отформатирован в **EXT4**  

![Форматирование XFS и EXT4](Screenshot_13.png){ #fig:005 width=70% }

## Монтирование файловых систем

- Вручную смонтирован **/dev/sdb5** в **/mnt/tmp**
- Добавлены записи в **/etc/fstab** для автоматического монтирования  

![Монтирование файловых систем](Screenshot_16.png){ #fig:006 width=70% }

## Самостоятельная часть

- На **/dev/sdc** созданы два GPT-раздела по 300 МиБ  

![Создание разделов ext4 и swap на GPT-диске](Screenshot_18.png){ #fig:018 width=80% }

## Самостоятельная часть

- **/dev/sdc2** — файловая система **EXT4**  
- **/dev/sdc3** — область подкачки **swap**  

![Форматирование разделов ext4 и swap](Screenshot_19.png){ #fig:019 width=80% }

## Самостоятельная часть

- Добавлены записи в **/etc/fstab**  

![Редактирование файла /etc/fstab для автоматического монтирования](Screenshot_20.png){ #fig:020 width=80% }

## Самостоятельная часть

- Проверка показала успешное монтирование и активацию swap

## Вывод

В ходе лабораторной работы освоены:
- создание разделов MBR и GPT;
